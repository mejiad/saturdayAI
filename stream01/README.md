# Stream

### Introducción

Este programa tiene el objetivo de mostrar el uso de Flux para la generación de respuestas del modelo LLM.

Para probar este programa una vez compilado y ejecutado es necesario hacer la llamada con el la ayuda del programa HTTPie, con el siguiente comando:


`http :8080/stream -b --stream --pretty none



Requerimientos:

Es necesario tener Ollama ejecutando en modo serve

$ ollama serve
